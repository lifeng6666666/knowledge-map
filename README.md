# 知识地图 (Knowledge Map)

一个可视化的知识图谱工具，支持在浏览器中创建、编辑和管理「学科 / 能力 / 课程」的关系网络。

**当前数据：107 个节点 | 143 条关系 | 12 个学科概念 | 12 个能力概念**

## 功能特性

- 📊 **网状图可视化**：使用 vis.js 绘制动态网络图
- ✏️ **图形化编辑**：无需代码，直接在界面上添加节点和关系
- 📚 **丰富的节点信息**：支持课程描述、推荐教科书、推荐课程、推荐项目
- 🔗 **简化关系类型**：仅保留「包含」与「前置」
- 💾 **数据管理**：
  - 本地存储（自动保存）
  - JSON 导入/导出
  - 下载为 JSON 文件
- 🎨 **美观的 UI**：深色侧边栏 + 网络可视化区域

## 快速开始

### 方式 1：本地使用

1. **克隆仓库**
   ```bash
   git clone https://github.com/yourusername/knowledge-map.git
   cd knowledge-map
   ```

2. **打开在浏览器**
   - 直接打开 `index.html`，或
   - 使用本地服务器：`python -m http.server 8000`
   - 访问 `http://localhost:8000`

## 使用指南

### 快速导入示例数据

打开应用后，你会看到预设的示例数据已加载，覆盖：
- 学科（数学与统计、计算机与信息技术、自然科学、生命科学、健康与医学等）
- 能力（学习能力、信息素养、写作能力、演讲能力、沟通能力等）
- 每门课程均附顶级书籍/课程/项目建议

如果你想使用自定义知识地图数据：
1. 点击左侧 "数据" 标签
2. 点击 "⬆️ 导入 JSON"
3. 打开你的 JSON 文件
4. 复制全部内容到对话框
5. 点击 "导入"

### 了解数据结构

详见 **KNOWLEDGE_MAP_STRUCTURE.md** 文件，包含：
- 完整的节点和关系说明
- 学习路径示例
- 如何扩展知识地图

### 添加节点

1. 在左侧面板选择节点类型：📖 课程 或 🎯 概念
2. 填写节点名称（必填）
3. 可选：添加描述、教科书、推荐课程、推荐项目
4. 点击 "✨ 添加节点"

**节点类型**：
- **课程节点**（蓝色正方形）：代表具体的课程
- **概念节点**（菱形）：代表学科或能力，可关联多个课程

### 添加关系

1. 选择 "从节点" 和 "到节点"
2. 选择关系类型：
   - **包含关系** 🟢：概念 → 课程
   - **前置课程** 🔴：课程 → 课程
3. 点击 "🔗 添加关系"

### 编辑和管理

- **查看详情**：点击任意节点，右侧面板显示详情
- **删除节点**：选中节点，点击"🗑️ 删除选中节点"
- **删除关系**：选中关系，点击"✂️ 删除选中关系"
- **交互操作**：
  - 拖拽节点移动
  - 滚轮缩放
  - 右键拖拽平移

### 数据管理

**导出数据**：
- 点击"数据"标签页
- 点击"⬇️ 导出 JSON"

**导入数据**：
- 点击"⬆️ 导入 JSON"
- 粘贴 JSON 数据
- 点击"导入"

**自动保存**：
- 所有更改自动保存到浏览器本地存储
- 刷新页面数据不会丢失

## JSON 数据格式

```json
{
  "nodes": [
    {
      "id": "node_1",
      "label": "微观经济学原理",
      "type": "course",
      "description": "研究个体经济主体...",
      "books": "教科书1\n教科书2",
      "courses": "推荐课程1\n推荐课程2",
      "projects": "项目1\n项目2"
    }
  ],
  "edges": [
    {
      "id": "node_1_node_2",
      "from": "node_1",
      "to": "node_2",
      "label": "包含",
      "type": "contains"
    }
  ]
}
```

## 常见问题

**Q: 数据会丢失吗？**
A: 不会。所有修改自动保存到浏览器的 localStorage。你也可以定期导出 JSON 备份。

**Q: 如何与他人分享？**
A: 
1. 导出数据为 JSON
2. 发送 JSON 文件给他人
3. 他人通过"导入 JSON"功能导入

**Q: 能自定义节点样式吗？**
A: 可以。编辑 `app.js` 中的颜色配置部分修改节点和边的样式。

**Q: 能导出为图片吗？**
A: 可以使用浏览器的截图功能或按 F12 打开开发者工具截图。
