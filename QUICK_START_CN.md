## 🎮 交互指南

### 图表操作
| 操作 | 效果 |
|------|------|
| 拖拽节点 | 重新排列位置 |
| 点击节点 | 在右侧显示详情 |
| 滚轮 | 缩放视图 |
| 方向键 | 平移视图 |
| 右上角按钮 | 导航控制 |

### 标签过滤
```
点击 [计算机科学] → 仅显示CS及其课程
点击 [机器学习AI] → 仅显示ML学习路径
点击 [全部] → 恢复完整知识地图
```

### 编辑操作
左侧边栏：
- **编辑器** - 添加节点和关系
- **详情** - 查看选中节点信息
- **数据** - 导入/导出/下载JSON

## 🎨 视觉标识

### 节点形状
- **钻石 ♦** - 概念节点
- **方形 ■** - 课程节点

### 边的颜色含义
- **红色 —** 前置关系（必须先学）
- **浅绿色 —** 相关关系（相关支撑）

## 🆘 常见问题

**Q: 怎样只看某个学科的课程？**  
A: 点击该学科的标签按钮，图表会自动过滤显示相关节点。

**Q: 我的编辑会被保存吗？**  
A: 是的，自动保存到浏览器localStorage。也可以JSON导出做备份。

**Q: 可以添加自己的课程吗？**  
A: 可以。左侧"编辑器"标签页中添加新节点和关系。

**Q: 如何恢复到原始数据？**  
A: 刷新页面或清除浏览器缓存，然后点击"全部"标签。

**Q: 支持哪些文件格式？**  
A: 目前支持JSON格式的导入导出。
